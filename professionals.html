Silex for teams

This is a whole new philosophy we have invented here.

We work in teams with Silex since 2006 like this: the developer provides the designer with components which take advantage of all the design features of the drag and drop editor. As a developer you have the tools to hack Silex in no time and add custom client side or server side components, which will then be available to the designer who knows how s.he wants the site to look like and how it is supposed to react to the user interactions. The designer can add a custom component to a site and style it and change its behaviour as needed
